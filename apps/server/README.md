# Agent Group Server

This private workspace package runs the Agent Group local server and bundled web application. It is
distributed only as part of the desktop application and is not published to npm.

Agent Group can execute local commands and modify project files through the selected coding agent.
Review the [project security policy](https://github.com/beileng1998/agent-group/blob/main/SECURITY.md)
before exposing the server remotely or enabling Full Access mode.

Source is maintained in the
[Agent Group repository](https://github.com/beileng1998/agent-group).
